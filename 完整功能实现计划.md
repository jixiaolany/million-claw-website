## 数据库设计

### 用户表 (users)
- id (主键)
- email (邮箱)
- invite_code (邀请码)
- subscription_status (订阅状态)
- created_at (创建时间)
- updated_at (更新时间)

### 邀请码表 (invite_codes)
- code (邀请码，主键)
- created_by (创建者)
- used_by (使用者)
- used_at (使用时间)
- is_active (是否有效)
- created_at (创建时间)

### 下载记录表 (downloads)
- id (主键)
- user_id (用户ID)
- platform (平台：windows/macos)
- version (版本号)
- downloaded_at (下载时间)
- ip_address (IP地址)

### 支付记录表 (payments)
- id (主键)
- user_id (用户ID)
- amount (金额)
- currency (货币)
- status (支付状态)
- paid_at (支付时间)
- subscription_period (订阅周期)

## API接口设计

### 认证相关
- `POST /api/auth/validate-invite` - 验证邀请码
- `POST /api/auth/login` - 用户登录
- `POST /api/auth/logout` - 用户登出
- `GET /api/auth/status` - 获取用户状态

### 下载相关
- `GET /api/download/check-permission` - 检查下载权限
- `POST /api/download/record` - 记录下载
- `GET /api/download/stats` - 下载统计

### 管理相关
- `GET /api/admin/users` - 获取用户列表
- `POST /api/admin/invite-codes` - 生成邀请码
- `GET /api/admin/stats` - 获取统计信息
- `GET /api/admin/payments` - 获取支付记录

## 🚀 部署步骤

### 步骤1: 创建GitHub仓库
1. 打开 https://github.com/new
2. 填写仓库信息:
   - **Repository name**: `million-claw-website`
   - **Description**: `Million Claw - 完全复刻EasyClaw的AI助手部署平台`
   - **Public**: ✅ 选择公开
   - **Initialize this repository with**:
     - ☐ README (不要勾选)
     - ☐ .gitignore (不要勾选)
     - ☐ license (不要勾选)
3. 点击 **Create repository**

### 步骤2: 推送代码
创建仓库后，GitHub会显示推送命令:

```bash
git remote add origin https://github.com/jixiaolany/million-claw-website.git
git branch -M main
git push -u origin main
```

**或者直接运行一键推送脚本**:
双击 `一键推送.bat`

### 步骤3: 开启GitHub Pages
1. 进入仓库: https://github.com/jixiaolany/million-claw-website
2. 点击 **Settings** → **Pages**
3. 配置:
   - **Source**: Deploy from a branch
   - **Branch**: main → /(root)
4. 点击 **Save**

### 步骤4: 验证网站
等待1-2分钟，访问:
**https://jixiaolany.github.io/million-claw-website/**

## 🚀 **部署步骤：**

### **第一步：登录GitHub**
1. 访问 https://github.com
2. 登录你的账号：`jixiaolany`
3. 确保已登录

### **第二步：创建新仓库**
1. 点击右上角"+" → "New repository"
2. 仓库名称：`your-app-website`（可自定义）
3. 描述：你的软件服务网站
4. 选择：Public（公开）
5. 勾选：Initialize this repository with a README
6. 点击：Create repository

### **第三步：上传网站文件**
1. 进入新建的仓库
2. 点击"Add file" → "Upload files"
3. 拖拽或选择 `完整的index.html` 文件
4. 重命名为 `index.html`（如果已存在则覆盖）
5. 填写提交信息："Initial website deployment"
6. 点击"Commit changes"

### **第四步：开启GitHub Pages**
1. 进入仓库设置：Settings → Pages
2. 在"Source"部分选择：Deploy from a branch
3. 分支选择：main
4. 文件夹选择：/(root)
5. 点击：Save

### **第五步：获取访问地址**
1. 等待几分钟（GitHub Pages需要时间部署）
2. 刷新页面，看到绿色提示："Your site is published at..."
3. 复制网址，如：`https://jixiaolany.github.io/your-app-website/`
